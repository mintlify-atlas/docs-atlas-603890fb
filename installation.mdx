# Installation

Install Paragrafs using your preferred package manager:

<CodeGroup>

```bash npm
npm install paragrafs
```

```bash yarn
yarn add paragrafs
```

```bash pnpm
pnpm install paragrafs
```

```bash bun
bun add paragrafs
```

</CodeGroup>

## Requirements

Paragrafs requires:

- **Node.js**: Version 24.0.0 or higher
- **Bun**: Version 1.3.5 or higher (if using Bun)

<Info>
  Paragrafs is built with TypeScript and ships with full type definitions included.
</Info>

## Package details

- **Package name**: `paragrafs`
- **Module type**: ESM (ECMAScript modules)
- **Entry point**: `dist/index.mjs`
- **Type definitions**: `dist/index.d.mts`
- **License**: MIT
